# lua-builds

Static Lua binaries for Windows (x64 & x86), built with MinGW-w64.

Includes executables, static libraries, and source code.

This is a community-maintained repository. For the official Lua project, please visit [www.lua.org](https://www.lua.org/).

## Downloads

Download the latest artifacts from the [Releases](https://github.com/Clothwest/lua-builds/releases/latest) page.

| Architecture | Description | File |
| :--- | :--- | :--- |
| **x64** | 64-bit Windows | `lua-x.x.x-win64.zip` |
| **x86** | 32-bit Windows | `lua-x.x.x-win32.zip` |

## Usage

Extract the downloaded zip file. The executable files (`lua.exe`, `luac.exe`) are located in the **`bin`** directory.

You can add the `bin` directory to your system `PATH` variable to use Lua from the command line.
